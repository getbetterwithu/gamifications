## 작성 가이드

각 슬롯은 다음 형식을 따라주세요:

```markdown
### M01 — 닮음의 정의·기호 표기

**문제 본문**:
> (여기에 학생에게 보여줄 문제 본문)

**보기** (객관식인 경우):
- ① ...
- ② ...
- ③ ...
- ④ ...

**정답**: ②

**정답 해설** (정답 시 학생에게 보여줄 1~2줄):
> ...

**오답 해설** (오답 시 보여줄 1~2줄, 선택적):
> ...

**힌트** (재시도 시 보여줄 1~2줄, 선택적):
> ...
```

> 형식이 단답형(수치 입력)인 경우 "보기"는 생략하고 "정답"란에 정확한 답(예: `5:8`)을 적어주세요. 가능한 경우 허용 답안 여러 개도 표기 (예: `5:8` 또는 `5/8`).
