📐폴리곤의 학당 — 사각형의 시험

중2 4단원 · 비주얼노벨 + 문제풀이 게임

📖 줄거리

고대 그리스, 피타고라스 학당의 신입생 폴리곤. 첫날부터 학당과 건축가들이 분쟁 중인 설계도를 마주한다. 평행사변형·직사각형·마름모·정사각형 — 도형들의 조건과 성질을 하나씩 풀어내며 분쟁의 진실에 다가간다.

🎯 학습 목표
  • 평행사변형이 되는 조건과 성질을 판별할 수 있다
  • 직사각형·마름모·정사각형의 정의와 대각선의 성질을 이해한다
  • 여러 가지 사각형의 포함 관계와 조건 추가에 따른 변화를 설명할 수 있다
  • 실생활 맥락(놀이기구·풀밭 등)에 사각형 성질을 적용한다
예상 시간
30~40분
문제 수
12 + 1
결과물
PDF 풀이지
게임 시작하기
🔑 선생님용 안내 (운영 가이드 · 치트키)

🎮 게임 흐름

  • 총 12개 정규 문제 (메인 10 + 사이드 2) + 특별 문제 1개
  • 각 문제는 첫 시도 정답에만 점수 부여. 오답 시 힌트가 단계별로 노출되고, 3회 실패하면 정답이 공개됩니다.
  • 특별 문제(Scene 8.5 — "잠깐 멈추기")는 점수·통계 집계에서 제외. 단, 정답을 맞춰야만 다음으로 넘어갈 수 있음 (자세히는 아래 참고).
  • 게임 종료 시 학생이 "📄 풀이 제출하기 (PDF)" 버튼을 눌러 본인용 풀이지를 다운로드. 오답 문제들의 빈 풀이 공간이 자동으로 들어가 있어 인쇄 후 손으로 다시 풀게 할 수 있습니다.
  • 엔딩은 첫 시도 정답 개수에 따라 3가지 분기 (학생에게는 엔딩 이름·등급 미공개).

📝 특별 문제 (Scene 8.5) 정답

게임 중간(Scene 8 다음)에 탈레스가 "지난 3주 동안 너희를 가르쳐주신 두 분 수학교생선생님의 성함"을 묻습니다.

정답: 남자선생님 황인수 / 여자선생님 김시원

점수에는 들어가지 않지만 이 문제를 통과해야만 다음 진도로 넘어갑니다. 학생이 모르면 안내해주세요.

🛠 ADMIN 모드 (선생님 전용)

  • 진입 1: ` (백틱) 키를 1.5초 이내에 3번 누르기
  • 진입 2 (백업): Ctrl+Shift+A
  • 진입 후 화면 상단에 🛠 ADMIN 뱃지와 ◀ 이전 문제 / 다음 문제 ▶ 버튼이 나타납니다.
  • 문제를 건너뛰거나 특정 장면으로 점프할 수 있어 수업 시연·디버깅에 사용. 같은 단축키로 다시 토글하면 해제.

키보드 한글 입력 상태에서도 작동합니다. 입력창에 포커스가 있으면 무시됩니다.

🔄 "다시 시작" 암호

엔딩 화면의 "다시 시작" 버튼을 누르면 암호 입력창이 뜹니다.

암호: 허철호짱

이 버튼은 학생이 PDF 풀이를 한 번 제출한 뒤에야 활성화되고, 누르면 점수·풀이·문제 결과가 전부 초기화됩니다. 학생 실수 방지용이니 필요할 때만 알려주세요.

💡 운영 팁

  • 크롬북에서 바로 실행 가능 — 별도 설치 없이 브라우저만 있으면 됩니다.
  • 학생이 중간에 종료해도 진행 상태(점수·풀이·문제 결과)가 LocalStorage에 저장되어 "이어서 하기"가 가능합니다.
  • "다시 시작" 버튼은 학생이 PDF로 풀이를 한 번 제출한 뒤에만 활성화됩니다. (실수로 점수가 날아가는 것 방지)
  • PDF는 학생 이름을 입력받아 파일명에 포함시킵니다. 인쇄 또는 LMS 제출이 모두 가능.
