# 4단원 시나리오 의뢰용 컨텍스트 패키지

> 이 파일은 별도 클로드 채팅에 그대로 복사·붙여넣어 시나리오를 의뢰하기 위한 자료입니다.

````markdown
# 게임 시나리오 작성 의뢰 (중2 수학 4단원 — 사각형 파트)

## 부탁드릴 일
중학교 2학년 수학 4단원(삼각형과 사각형의 성질) 중 **사각형 파트**(평행사변형·직사각형·마름모·정사각형·등변사다리꼴) 복습용 비주얼노벨 + 육성 시뮬레이션 게임 시나리오를 작성해주세요.

**이미 만든 5단원 게임("시밀러의 학당 — 닮음의 신전")의 후속편**으로, **세계관과 등장인물은 그대로 재활용**하되 **사건은 완전히 새로** 만듭니다. (마블 시네마틱 유니버스가 같은 등장인물로 다른 영화를 만드는 방식과 같습니다.)

**중요**:
1. 실제 수학 문제·사이드 문제의 본문은 시나리오에 포함하지 마세요. "이 장면에서 [난이도 중] 평행사변형 조건 문제 1개가 필요함" 같은 **문제 슬롯(blank)** 형태로 표시.
2. **문제는 교과서 405·406 소단원의 "바로 확인하기"를 그대로 사용**할 예정이므로, 시나리오에서 다룰 슬롯은 그 12개 문제와 매칭되어야 함 (아래 6번 항목 참조).
3. **새 평가 시스템**을 시나리오에도 반영 — 오답 시 정답 낼 때까지 재시도, 매 오답마다 힌트 단계별 표시. 캐릭터 톤이 이 흐름과 어울려야 함.

## 1. 프로젝트 개요

### 게임 형식
- 비주얼노벨 + 육성 시뮬레이션 (프린세스메이커 스타일)
- HTML/CSS/JS 단일 웹앱 (크롬북에서 바로 실행)
- 플레이 시간: 약 30분
- 인원: 한 반 24명, 개인 플레이
- 활용 시점: **4단원 마무리 복습** (2주 후 사용 예정)

### 5단원과의 관계 (MCU 방식)
| 요소 | 5단원 | 4단원 |
|---|---|---|
| 세계관 | 피타고라스 학당 | **그대로** |
| 캐릭터 5명 | 시밀러·피타고라스·탈레스·다이달로스·이리스 | **그대로** |
| 캐릭터 5표정 | neutral / smile / surprised / thinking / sad | **그대로** |
| 사건 | 황금 비례판 실종 사건 (닮음으로 추적) | **완전 새로** (사각형 관련 사건) |

## 2. 학습 대상 — 4단원 사각형 파트

### 404. 평행사변형의 성질 (배경 지식 — 학습 맥락에 녹임)
- 평행사변형 기호: □ABCD
- 대변·대각 개념
- **3대 성질**:
  ① 두 쌍의 대변의 길이 각각 같음
  ② 두 쌍의 대각의 크기 각각 같음
  ③ 두 대각선은 서로 다른 것을 이등분

### 405. 평행사변형이 되는 조건 (문제 출처)
- **5가지 조건** (사각형이 평행사변형이 되는):
  ① 두 쌍의 대변이 각각 평행 (정의)
  ② 두 쌍의 대변의 길이가 각각 같음
  ③ 두 쌍의 대각의 크기가 각각 같음
  ④ 한 쌍의 대변이 평행 + 그 길이가 같음
  ⑤ 두 대각선이 서로 다른 것을 이등분
- 함정 포인트: "두 대각선의 길이가 같음"은 평행사변형 조건이 아님 (직사각형 조건임)
- 실생활: 빙글빙글 놀이기구 (수평 유지 원리)

### 406. 여러 가지 사각형의 성질 (문제 출처)
- **직사각형**: 두 대각선 길이 같고, 서로 다른 것 이등분
- **마름모**: 두 대각선이 서로 다른 것을 수직이등분
- **정사각형**: 두 대각선 길이 같고, 서로 다른 것 수직이등분 (직사각형 + 마름모)
- **등변사다리꼴**: 평행하지 않은 두 변 길이 같음, 두 대각선 길이 같음
- 사각형 사이의 관계도: 사다리꼴 → 평행사변형 → 직사각형/마름모 → 정사각형
- 실생활: 스마트폰 화면, 사방치기, 연, 바리케이드 등

## 3. 세계관 및 캐릭터 (5단원과 동일)

### 세계관
**고대 그리스, 피타고라스 학당** — 케팔로니아 섬에 위치. 5단원 사건(황금 비례판 실종) 이후의 시점.

### 캐릭터 5명

| 영문 키 | 이름 | 5단원 역할 | 4단원에서 활용 방향 |
|---|---|---|---|
| `apprentice` | 시밀러 (주인공) | 14세 견습생, 호기심 多 | 그대로 — 정식 제자로 한 단계 성장한 상태 |
| `pythagoras` | 피타고라스 | 60대 스승, 위엄·엄격 | 그대로 — 다만 사건 풀어내는 데 좀 더 신뢰 |
| `thales` | 탈레스 | 40대 친근 멘토, 역사 일화 담당 | **힌트 화자** 역할 ★ (오답 시 학생에게 힌트 전달) |
| `rival` | 다이달로스 | 14세 라이벌, 자신감·거만 | 5단원 마지막에 살짝 협력적이 됨 → 이번엔 더 자연스러운 동료/경쟁 |
| `companion` | 이리스 | 14세 동기, 차분·사려깊음 | 그대로 — 통찰 담당 |

각 캐릭터 5표정 사용 가능: neutral / smile / surprised / thinking / sad

## 4. 사건 — 사각형 파트에 어울리는 새 사건 (예시·자유롭게)

### 사건 후보 (선택 또는 변형 가능)
- **A. 신전 건축 설계도 분쟁**: 학당에서 새 신전을 짓는데, 설계도의 사각형들이 어떤 종류인지 분류하다가 분쟁이 생김. 평행사변형인지 직사각형인지 마름모인지 판별해야 함.
- **B. 사라진 보석 상자**: 정사각형 모양의 보석 상자가 마름모 모양으로 변형되어 보관실 어딘가에 숨겨짐. 사각형의 변화 관계를 추적.
- **C. 시장의 측량 의뢰**: 도시 광장의 땅이 사각형인데 정확한 종류를 알아야 평등하게 나눌 수 있음.

→ **A 추천** (학습 맥락이 가장 자연스러움). 다만 작가님 판단에 맡깁니다.

### 사건의 결말
사각형의 분류·관계를 모두 익혀서 사건 해결. 5단원처럼 분기 엔딩 3개 (우수·평범·낙제).

## 5. 새 평가 시스템 (시나리오에 반영해야 할 부분)

5단원과 다른 평가 방식이 도입됐습니다. 시나리오의 캐릭터 대사 톤이 이 흐름과 어울려야 합니다.

### 평가 규칙
1. **첫 시도 정답만 점수 부여** (재시도로 맞춰도 점수 0)
2. **오답이어도 진행 X** — 정답 낼 때까지 재시도
3. **매 오답마다 힌트 단계별 표시** (탈레스 캐릭터 톤으로)
4. **3회 모두 실패 시 정답 공개 + "미해결" 기록**
5. **최종 제출 PDF에 재과제 대상 명단** (미해결 + 수정 완료 분리)

### 캐릭터 대사 톤 가이드
- 정답 시: 평소 캐릭터 톤 그대로 (피타고라스 "옳다" / 탈레스 "역시!" 등)
- **오답 시 (시나리오에는 작성 X — 게임 엔진이 자동 처리)**: 힌트는 별도 hints 배열로 작가님이 만들지 않습니다. **선생님이 채워넣을 예정**.
- 단, 시나리오의 분위기는 **"틀려도 다시 도전하는 게 정상"** 톤을 깔아주세요. 시밀러가 처음에 실수해도 동료들이 격려하는 분위기.
````
