# 중학교 2학년 4단원 — 삼각형과 사각형의 성질

> 단원 마무리 복습용 게임 (비주얼노벨 + 육성 시뮬레이션)

---

## 단원 구성

4단원은 6개 소단원으로 이루어짐:

- **401** 이등변삼각형과 직각삼각형
- **402** 삼각형의 외심
- **403** 삼각형의 내심
- **404** 평행사변형의 성질
- **405** 평행사변형이 되는 조건
- **406** 여러 가지 사각형의 성질

이 단원의 첫 번째 게임은 **`stage1_quadrilaterals`** — 사각형 관련 소단원(404~406) 중심.

> 참고: 5단원은 도형의 닮음 5개 소단원을 3개 스테이지로 나눴지만, 4단원은 이번 한 번만 만들어 단원 마무리 복습에 사용 (2주 후 예정).

### 2. 시나리오 의뢰
세계관(피타고라스 학당) + 인물(시밀러·피타고라스·탈레스·다이달로스·이리스) 그대로 재활용.
사건은 새로 — 사각형 관련 미스터리나 사건으로.
별도 클로드 채팅에 의뢰 (PDF 받은 후 컨텍스트 패키지 자동 생성 예정).

### 3. 시나리오 받아오면 게임 코드 작업
5단원 `stage1_similarity`의 엔진을 그대로 가져와서 시나리오·소품·배경만 4단원 내용으로 교체.

## 5단원과의 관계

| 요소 | 4단원 처리 |
|---|---|
| 게임 엔진 (필기·PDF·UI) | 5단원 코드 100% 재활용 |
| 캐릭터 5명 + 25표정 | 동일 (마블 MCU 방식 — 같은 인물·다른 사건) |
| 배경 SVG/이미지 | 일부 재활용 + 신규 (사각형 단원 특화) |
| SVG 소품 | 신규 (외심·내심·평행사변형 등) |
| 시나리오 | 신규 |
| 문제 슬롯 | 신규 (4단원 내용으로) |

4단원 완성 → 그 경험으로 시스템 전체 개선 → 5단원에도 역적용 예정.
