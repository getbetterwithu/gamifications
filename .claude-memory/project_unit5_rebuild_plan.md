# 5단원 (닮음) 시나리오 전면 재구성 계획

**결정일**: 2026-05-12
**현재 상태**: 5단원 홀드 — 4단원 완성 우선

## 핵심 결정
4단원 시나리오가 먼저 나오고 5단원이 그 후속이 되는 시간 흐름으로 재정렬됨. 5단원은 부분 수정이 아니라 **완전히 갈아엎는다**. 사용자가 5단원 소단원 "바로 확인하기" 문제를 제공하면 그걸 기반으로 새 시나리오를 짠다 (4단원과 동일한 방식).

**Why**: 4단원이 폴리곤의 입학 첫날 사건으로 자리잡았고, Scene 12에 별명 예고·황금 비례판 떡밥이 박혔음. 기존 5단원은 단독작으로 만들어져서 4단원과의 연결이 약함. 부분 수정(Level 2~3)보다 새로 짜는 게 더 자연스럽다고 사용자가 판단.

**How to apply**:
- 4단원 게임 완성 전까지는 5단원 손대지 말 것
- 4단원 완성 → 사용자가 5단원 소단원 문제 제공 → 그걸 보고 새 시나리오 작성 (외부 채팅 의뢰 또는 직접)
- 5단원도 4단원처럼 교과서 소단원 "바로 확인하기" 문제를 그대로 사용

## 작업 순서
1. (지금) 4단원 게임 완성 — 시나리오 코드화 + 문제 박스 렌더 버그 수정
2. 4단원 완성 후 — 사용자가 5단원 소단원 문제 제공
3. 5단원 시나리오 새로 짜기
4. 5단원 코드화

## 4단원 연결 훅 처리
4단원 Scene 12에 박힌 떡밥은 새 5단원에서 자연스럽게 회수:
- **별명 예고** — 피타고라스가 "닮음 다룰 때 어울리는 이름 얹어주마" 약속 → 새 5단원 초반에 "시밀러" 별명 부여 장면
- **황금 비례판 암시** — 4단원에서 호기심만 남긴 상태 → 새 5단원에서 회수 (단, 새 사건과 어떻게 엮을지는 새 시나리오 작성 시점에 판단)
