# 4단원 문제 정답 검수 작업 (보류)

**기록일**: 2026-05-13
**상태**: 보류 — 사용자 요청, 향후 작업

## 배경
사용자가 4단원 12개 문제(M01~M10, S01, S02)를 만들 때 **교과서 405·406 소단원 "바로 확인하기" 출처만** Claude에게 알려주었고, Claude가 PDF에서 텍스트와 도형을 추출하면서 **정답까지 직접 계산해 PROBLEMS 객체에 채워넣었음**. 따라서 정답이 부정확할 가능성 있음.

**사용자 요청**: "지금 문제의 답을 정확하게 모르고 있는거 같아. 나중에 문제의 답만 따로 정리할 수 있는 검수과정을 넣어줘."

## How to apply
4단원 게임이 어느 정도 안정되면, **정답 검수 전용 화면 또는 문서**를 만든다. 형태 후보:

### A. 정답 검수 전용 HTML
- 12개 문제를 한 페이지에 나열
- 각 문제: 본문 + 도형 + 현재 Claude가 입력한 정답 + 풀이 과정 메모
- 사용자가 한눈에 보고 ❌/✓ 표시 → 수정 요청 가능
- `storyboard.html`과 별도 페이지로

### B. 정답 검수 markdown 문서
- `unit4_tri_quad/docs/answer_review.md`
- 각 문제 — 본문 / 도형 참조 / 정답 / 풀이 설명
- 사용자가 코멘트 달거나 직접 수정

### C. 둘 다
- HTML로 빠르게 훑고, markdown으로 수정 사항 기록

**추천**: A (HTML) — 도형 이미지·문제 본문을 한눈에 보면서 검수 가능

## 작업 시점
4단원 게임의 다른 부분(시나리오·배경·게임 흐름)이 안정된 후. 사용자가 "정답 검수하자" 신호를 줄 때 진입.

## 검수 대상 12개 (정답이 정확한지 사용자 검토 필요)
- M01 (OX): "두 쌍의 대변 길이 같음" → O
- M02 (객관식 다답): 4개 그림 중 평행사변형 → (1), (3) — [0, 2]
- M03 (단답): 평행사변형 ∠x, ∠y → 60°, 120°
- M04 (OX): "두 대각선 길이 같음 = 평행사변형" → X
- M05 (단답): 평행사변형 조건 x, y → 35, 4
- M06 (객관식): AB=DC + 추가 조건 → ? (정답 확인 필요)
- M07 (OX): "수직 대각선 + 직사각형 = 정사각형" → O
- M08 (단답): 직사각형 x, y → ?
- M09 (단답): 마름모 △AOB 넓이 → ?
- M10 (단답/객관식): 조건 만족 사각형 이름 → ?
- S01 (OX): 놀이기구 평행사변형 수평 → O
- S02 (단답): 정사각형 풀밭 목장 넓이 → ?

→ 특히 단답형(M03, M05, M08, M09, M10, S02)과 M06은 Claude의 계산이 맞는지 확실치 않음. 검수 화면 만들 때 풀이 과정도 함께 노출해서 사용자가 확인할 수 있게 할 것.
